What the ML model learns from real data

Real training data

128 matches from 2018 (Russia) and 2022 (Qatar) World Cups. Every result feeds an Elo engine that tracks team strength dynamically the model learns from how those Elo gaps predicted real outcomes.

Gradient descent · live

200 epochs of mini-batch gradient descent run right here in your browser. The loss curve you see is the model actually converging weights w₁, w₂, w₃ are learned, not set by hand.

Logistic regression win probability formula
P(A beats B) = σ( w₁·ΔElo + w₂·knockout_stage + w₃·bias )
σ(x) = 1/(1+e−x)  |  Loss = −∑ y·log(p) + (1−y)·log(1−p)  |  Optimiser: mini-batch GD, lr=0.05
